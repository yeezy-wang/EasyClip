# EasyClip — 设计规范

## 色彩系统

EasyClip 使用系统动态色，自动适配浅色/深色模式。

| 名称 | 浅色模式 | 深色模式 | 用途 |
|------|---------|---------|------|
| Primary | `#2563EB` | `#2563EB` | 强调色、选中态、按钮 |
| Primary Light | `#E0E7FF` | — | 已复制标签背景 |
| Danger | `#EF4444` | `#EF4444` | 删除按钮、清除按钮 |
| Background | `NSColor.windowBackgroundColor` | 面板/卡片背景 |
| Text Primary | `NSColor.labelColor` | 主文字 |
| Text Secondary | `NSColor.secondaryLabelColor` | 时间戳、辅助文字 |
| Border | `NSColor.separatorColor` | 卡片边框 |
| Search BG | `NSColor.controlBackgroundColor` | 搜索框背景 |
| Hover BG | `NSColor.quaternaryLabelColor` | 卡片 hover 背景 |

## 组件规格

### 文字卡片 (HStack)

```
┌──────────────────────────────────────────┐
│  📋  这是复制的内容预览最多三行文字...      │
│      25.05.16 14:30               ⬆️ 🗑️  │
└──────────────────────────────────────────┘
```

- 左侧: 📋 图标 36×36px
- 中间: 正文 13pt × 最多 3 行 + 时间 11pt
- 右侧 (hover 显示): ⬆️ 13pt + 🗑️ 10pt, 间距 6px
- 已复制状态: 显示 "已复制" 蓝色标签

### 图片卡片 (VStack)

```
┌──────────────────────────────────────────┐
│                                          │
│          [160px 缩略图预览]               │
│                                          │
│  来自"微信"         25.05.16 14:30  👀⬆️🗑️│
└──────────────────────────────────────────┘
```

- 顶部: 缩略图 160px 高, .fit 自适应, 圆角 4px
- 底部: 来源应用 13pt + 时间 11pt + 操作按钮 (hover)
- 操作按钮: 👀 10pt (仅图片), ⬆️ 13pt, 🗑️ 10pt

### 搜索框

```
┌──────────────────────────────────────────┐
│  🔍  搜索剪贴板（仅支持文本搜索）          │
└──────────────────────────────────────────┘
```

- 仅在有记录时显示
- 占位文字 "搜索剪贴板（仅支持文本搜索）"

### 底部栏

```
┌──────────────────────────────────────────┐
│  N 条记录        清除数据（不含已置顶内容）│
└──────────────────────────────────────────┘
```

- 左侧: "N 条记录" 12pt 灰色
- 右侧: "清除数据（不含已置顶内容）" 12pt 红色
- 点击后内联变为 "确认清除？ [确定] [取消]"
- 无模态弹窗，不破坏 popover 事件链

### 设置窗口

```
┌──────────────────────────────────────────┐
│  EasyClip - 设置                     ⚙️  │
│  ─────────────────────────────────────── │
│  保留天数                                │
│  [ 1 天 │ 3 天 │ 5 天 ]                 │
│  开机启动                      ● 已开启  │
│  ─────────────────────────────────────── │
│  数据管理                    清空所有数据 │
│  ─────────────────────────────────────── │
│  应用                           关闭应用 │
└──────────────────────────────────────────┘
```

- 窗口: 300×300px
- 保留天数: 自定义段选按钮, 选中态蓝色填充
- 开机启动: 圆点指示器 + 文字
- 关闭应用: 二次确认弹窗

### 空状态

```
┌──────────────────────────────────────────┐
│                                          │
│              📋                          │
│       暂无剪贴板历史                      │
│    复制文字或图片即可开始                   │
│                                          │
└──────────────────────────────────────────┘
```

- 无搜索: 📋 + "暂无剪贴板历史" + "复制文字或图片即可开始"
- 有搜索无结果: 🔍 + "无匹配结果" + "尝试其他关键词"

## 菜单栏图标

- 来源: 自定义 PNG 图片 (menubar_icon.png)
- 尺寸: 21×21px
- 模板图: `isTemplate = true` (自动适配浅色/深色菜单栏)

## 应用图标

- 来源: 自定义 PNG → icns
- 格式: AppIcon.icns
- 尺寸: 16/32/128/256/512 @1x & @2x

## 动画

| 场景 | 动画 |
|------|------|
| 面板展开 | NSPopover animates=true |
| 卡片 hover | 背景色 0.15s easeOut |
| 清空确认 | 底部栏 inline 切换 |
| 已复制反馈 | 标签显示 1.2s 后消失 |

## 深色模式

- 所有背景/文字色使用 `NSColor` 动态色
- Primary/Danger 色值不变
- 菜单栏图标使用 `isTemplate = true` 自动适配
