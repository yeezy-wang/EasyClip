> **版本:** 1.0

## 1. 产品概述

### 1.1 产品定义

EasyClip 是一款 macOS 菜单栏剪贴板历史管理工具。自动记录用户复制的文字和图片内容，提供搜索、置顶、保留策略、Quick Look 预览和回写粘贴功能，帮助用户高效管理和复用剪贴板内容。

### 1.2 产品背景

日常使用电脑时，用户频繁进行复制粘贴操作，但系统剪贴板只保留最近一次内容。当用户需要找回之前复制的内容时，不得不重新查找源文件并重新复制，效率低下。EasyClip 通过自动记录和管理剪贴板历史，解决这一痛点。

### 1.3 产品目标

| 目标 | 指标 |
|------|------|
| 提升复制粘贴效率 | 用户可在 2 秒内找到并回写历史内容 |
| 无感运行 | 菜单栏常驻，不占用 Dock 空间，不打断工作流 |
| 数据安全 | 本地存储，不联网，不上传任何数据 |
| 系统兼容 | 支持 macOS 10.15 (Catalina) 及以上版本 |

## 2. 目标用户

### 2.1 用户画像

| 角色 | 特征 | 使用场景 |
|------|------|----------|
| 开发者 | 频繁复制代码片段、命令行、配置信息 | 快速回找之前复制的代码或命令 |
| 写作者 | 复制参考资料、引文、数据 | 管理素材收集，避免重复查找 |
| 设计师 | 复制颜色值、图片素材 | 保存常用色值和设计素材 |
| 普通用户 | 复制文字、图片用于聊天/文档 | 找回误覆盖的复制内容 |

### 2.2 核心需求

1. **快速访问** — 菜单栏一键打开，不切换窗口
2. **内容找回** — 快速搜索和定位历史复制内容
3. **一键回写** — 点击即复制到剪贴板，直接粘贴使用
4. **隐私安全** — 数据仅本地存储，不上传云端

## 3. 功能需求

### 3.1 自动记录剪贴板

**用户故事:** 作为用户，我希望系统自动记录我复制的所有文字和图片，无需手动操作。

| 编号 | 需求描述 | 验收标准 |
|------|----------|----------|
| F-1.1 | 监听系统剪贴板，自动记录文字内容 | 复制文字后 ≤1 秒内出现在历史列表 |
| F-1.2 | 监听系统剪贴板，自动记录图片内容 | 复制图片后 ≤1 秒内出现，保存原图质量 |
| F-1.3 | 复制文件时不产生记录 | 从 Finder 复制文件不记录 |
| F-1.4 | 从历史面板复制内容不重复记录 | 点击卡片回写后不产生新记录 |
| F-1.5 | 相同文字内容自动去重 | 连续复制相同文字只保留一条 |

**实现细节:**
- 通过 0.5 秒轮询 `NSPasteboard.general.changeCount` 检测变化
- 检测 `.fileURL` 类型时跳过文件复制
- `skipNextPasteboardChange` 标志防止自复制循环

## 5. 数据模型

### 5.1 核心实体: ClipboardItem

| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | UUID | 是 | 主键 |
| timestamp | Date | 是 | 捕获时间，排序依据 |
| contentType | String | 是 | "text" / "image" |
| textContent | String? | 否 | 原始文字内容 |
| imageData | Binary | 否 | 原始图片字节 |
| thumbnailData | Binary | 否 | 缩略图 (480px PNG) |
| sourceApp | String? | 否 | 来源应用名 |
| isPinned | Boolean | 是 | 是否置顶 |

### 5.2 索引

- `timestamp` — 时间排序
- `isPinned` — 置顶筛选
- `textContent` — 文字搜索

### 5.3 排序规则

1. 置顶内容在前 (`isPinned DESC`)
2. 同级按时间降序 (`timestamp DESC`)

## 6. 设计约束

### 6.1 macOS 10.15 兼容性

由于最低支持 macOS 10.15，以下 API 不可用，需使用替代方案：

| 不可用 API | 替代方案 |
|-----------|---------|
| `@main` / `App` | `main.swift` + `NSApplicationDelegate` |
| `Image(systemName:)` | Unicode 字符图标 |
| `@AppStorage` | `UserDefaults` + `@State` |
| `@StateObject` | `@ObservedObject` |
| `LazyVStack` | `VStack` |
| `SMAppService` | LaunchAgent plist |

### 6.2 菜单栏交互

- 使用 AppKit `NSStatusBar` + `NSPopover(.transient)`
- 左键点击弹出面板，右键弹出菜单
- 不使用 SwiftUI `MenuBarExtra`（需 macOS 13+）

### 6.3 图片处理

- 原图保存：TIFF/PNG 原字节存入 Core Data
- 缩略图：480px PNG 无损生成，160px 显示
- 回写时：匹配原格式，不转换
