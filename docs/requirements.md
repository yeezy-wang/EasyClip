# EasyClip — 需求文档

## 概述

EasyClip 是一款运行在 macOS 上的菜单栏剪贴板历史管理工具。自动记录所有文字和图片复制内容，可设置保留天数，支持搜索、置顶、Quick Look 预览和回写粘贴。

## 功能需求

### 1. 自动记录剪贴板
- 监听系统剪贴板，自动记录所有文字和图片复制
- 图片保存原图质量，不做压缩
- 复制文件时不产生记录（仅记录文本和图片）
- 从历史面板复制内容不会重复记录

### 2. 菜单栏驻留
- 菜单栏图标常驻，不占用 Dock 空间 (LSUIElement)
- 左键点击图标展开历史面板
- 右键点击图标弹出菜单（设置 / 退出）
- 点击面板外部区域自动关闭

### 3. 卡片列表展示
- 所有记录按时间降序排列（最新在上）
- 文字卡片: 左侧图标 + 正文预览(最多3行) + 绝对时间
- 图片卡片: 大图缩略图(160px) + 来源应用 + 时间 + 预览按钮
- 时间格式: YY.MM.DD HH:mm (中文)

### 4. 置顶功能
- 点击 ⬆️ 置顶，点击 ⬇️ 取消置顶
- 置顶条目始终在最前面，独立分区显示
- 置顶条目受双重保护：全部清除和过期清理均不删除

### 5. 删除与清空
- 单条删除: hover 卡片后点击 🗑️
- 全部清除: "清除数据（不含已置顶内容）" 按钮 → 内联确认 → 仅清除非置顶项
- 设置中可清空所有数据（含置顶项）

### 6. 搜索功能
- 搜索框实时过滤文字内容
- 仅在有记录时显示搜索框
- 占位文案: "搜索剪贴板（仅支持文本搜索）"
- 关闭面板时自动清空搜索输入

### 7. 保留策略
- 默认保留 1 天，可选 1/3/5 天
- 超过保留天数的未置顶条目自动清理
- 启动时 + 每小时执行清理

### 8. 回写粘贴
- 点击卡片将内容写回系统剪贴板
- 文字回写为纯文本，图片回写为原图 (TIFF/PNG 类型匹配)
- 粘贴成功后显示 "已复制" 反馈标签 (1.2s)

### 9. 图片预览
- 图片卡片 hover 时显示 👀 预览按钮
- 空格键可快速预览当前 hover 的图片
- 使用 macOS 原生 Quick Look 预览窗口

### 10. 开机启动
- 设置页开关控制
- 通过 LaunchAgent 实现，重启后自动运行

### 11. 数据持久化
- Core Data 本地存储 (SQLite + 外部图片文件)
- 重启电脑保留全部历史
- 仅通过手动删除或过期策略清理数据

### 12. 深色模式
- 自动适配系统浅色/深色模式
- 使用 NSColor 动态色实现

## 非功能需求

- 原生 macOS 开发 (Swift + SwiftUI + AppKit)
- 最低兼容 macOS 10.15 (Catalina)
- 面板尺寸: 340 × 620px
- 响应式监听: 0.5s 轮询间隔
- 通用二进制: arm64 + x86_64
